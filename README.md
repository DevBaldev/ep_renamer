# Episode Renamer

A simple Python program that bulk renames episodes of anime.

## Table of Contents

- [Features](#features)
- [Installation](#installation)
- [Usage](#usage)
- [License](#license)

## Features

- Get user args for start, end and type of episodes.
- Rename episodes using [Yazi](https://github.com/sxyazi/yazi) bulk renamer.

## Installation

Follow these steps to install and run the project:

1. Clone the repository:

   ```bash
   git clone https://github.com/DevBaldev/ep_renamer.git
   ```

2. Navigate into the directory:

   ```bash
   cd ep_renamer
   ```

## Usage

Rename using Yazi, run the following command:

```bash
uv run main.py -s 1 -e 69 -t "mkv"
```

Copy the output and select all files in Yazi than paste.
